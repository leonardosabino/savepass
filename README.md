# Getting Started
1.	Installation process
  - Clone or download the project
  - .\gradlew build
  - docker-compose up
  - .\gradlew bootRun
3.	Software dependencies
  - JDK 11
  - Gradle
  - Docker
  - Docker Compose
4.	API references
  - localhost:8081/swagger-ui/
